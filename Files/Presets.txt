 Установка иконок делается по принципу "с хитростью":
 Всем дискам глобально заменяется иконка для USB, таким образом все диски,
 подключенные уже или потом, как угодно, будут иметь иконку флешки. В том числе и локальные диски.
 Кроме системного, так как у него принудительная иконка, и задачи ее сменить не стоит.
 А так как локальные и RAM диски стационарные, т.е. буква не меняется, то
 на основе дисков в системе заменяются иконки индивидуально под букву, для RAM дисков своя, для Локальных дисков своя.
 Индивидуальная иконка на букву имеет приоритет, по этому эти диски будут с нужной иконкой. Остальные все с иконкой USB.
 Другие иконки это не затрагивает, например, смартфоны будут со своей иконкой как обычно.

   Set-Icon-Disks-LOCAL ="%SystemRoot%\System32\imageres.dll,-32"=    Ставится для локальных дисков индивидуально на букву (стандартная иконка)
     Set-Icon-Disks-RAM ="%SystemRoot%\System32\imageres.dll,-34"=    Ставится для рам дисков индивидуально на букву (иконка микросхемы)
     Set-Icon-Disks-USB ="%SystemRoot%\System32\DDORes.dll,-2389"=    Ставится для всех дисков в системе глобально (иконка флешки)



 Обход блокировок сайтов провайдером через файл автонастройки proxy.pac
 Обход производится только для сайтов из списка в файле, остальные сайты открываются на прямую.
 В редких случаях, когда провайдер пускает клиентов через свой proxy, то для работы обхода через VPN
 необходимо дополнительно снять галочку "Автоматическое определение параметров":
 в ситемных свойствах браузера (IE) -> Подключения -> Настройка сети
 В батнике сделаны 2 предустановки только для России. Для других регионов надо добавить свой файл в этот параметр:
 например локально так: D:\proxy.pac, или удаленно так: https://www.test.ru/proxy.pac

    Set-UnBlock-Site-My =""=    Тут нужно указать путь к вашему файлу proxy.pac, локально или удаленно

 Изменение расположения папок Temp:
 Для решения проблем с кэшем Adobe, добавлен обратный слэш на конце путей папок Temp.
 Для программ и системы это указывает, что путь является папкой, а не файлом.
 Если вы не переносите, то можно к дефолтным параметрам добавить также. Хуже не будет.

   Set-User-Temp-Folder ="F:\Temp\"=                 Расположение папки temp пользователя
 Set-System-Temp-Folder ="%SystemDrive%\TempSys\"=   Расположение системной папки temp, не указывайте туже папку, что и для пользователей! Подробнее в Menu Info.txt

 Изменение расположения папок пользователя.
 Не ставьте в конце пути знак слэш: \, например так нельзя: "D:\"
 Примеры путей для папок:
 "D:" - Папки переместятся в корень диска "D"
 "Z:\MyFolder" - Папки переместятся на диск "Z" в папку "MyFolder"
 "C:\My\%UserName%" - Папки переместятся в "C:\My\папка с именем пользователя"

 Set-Dir-User-Folders   ="E:"=    Назначение расположения для переноса папок пользователя, кроме папки Загрузки
 Set-Dir-User-Downloads ="E:"=    Назначение расположения для переноса папки Загрузки
